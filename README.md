# 🎮 GamersHub

## 📌 Overview
**GamersHub** is an ASP.NET Core MVC web application for video game enthusiasts.  
It allows users to browse, search, filter, and manage a collection of games, platforms, and genres.  
The application includes authentication, role-based authorization, custom error pages, and database seeding.

---

## 🚀 Features
- **Game Management**
  - Browse, search, and filter games by title or genre.
  - View game details with images and release dates.
  - Pagination for smooth navigation.

- **Platform & Genre Management**
  - Full CRUD operations (Admin only).

- **User Accounts**
  - Identity-based authentication and authorization.
  - Admin and User roles with different permissions.

- **Error Handling**
  - Custom 404 (Not Found) and 500 (Server Error) pages.

- **Database Seeding**
  - Preloaded genres, platforms, roles, users, and sample games.

---

## 🛠️ Technologies Used
- **Backend:** ASP.NET Core MVC, C#
- **Database:** Entity Framework Core, SQL Server
- **Authentication:** ASP.NET Core Identity
- **Frontend:** Razor Pages, Bootstrap 5
- **Other:** LINQ, Dependency Injection, Middleware
